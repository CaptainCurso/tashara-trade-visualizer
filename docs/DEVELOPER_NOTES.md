# Recommended Next Iteration Targets

## 1. GM Snapshot Controls

Add a lightweight admin or local-only snapshot editor for:

- current day
- temporary price overrides
- faction status changes
- event toggles

This would let the public-facing app stay simple while giving you a much smoother campaign update workflow.

## 2. Sublocation Markets

Break big worlds into named market nodes such as:

- Zh'lani vs Free Current settlements on Qesh
- Keth-Saran vs Zur-Malak vs Thunderpeak Hold vs Tidecall Harbor on Tash-Ahn
- Surface Singers vs Middle Resonance contacts on Phorvaire

That would make the trade game much more expressive without changing the overall architecture.

## 3. Event and Faction Modifiers

Add an event layer that can shift:

- price multipliers
- route risk
- cargo availability
- local rules

Examples: parasite outbreaks on Qesh, water crisis changes on Tash-Ahn, piracy spikes between Vesha-Kai and Tash-Ahn.

## 4. Cargo Capacity and Operational Cost

Add ship-level constraints such as:

- cargo capacity
- crew upkeep
- hazard insurance
- route prep cost

This would make "profit per day" more realistic and support better party planning.

## 5. Service Economy Support

Several of the best Tashara opportunities are really service trades, not cargo trades.
The current code is ready for a later service-contract layer covering:

- transport to moons
- mediation contracts
- first-contact brokerage
- expedition support
- protection / escort work
